# 📢 InkNest v1.4.7 Release Announcement

Hey everyone! 🎉 We're excited to announce **InkNest v1.4.7** is now available!

This release focuses on reading continuity and source reliability. We've improved manga resume handling, refreshed ReadAllComics compatibility, and fixed image loading edge cases.

<!-- truncate -->

## ✨ What's New

#### Smarter Manga Resume Experience
- **Continue from your last read chapter** directly from Manga Details
- **Saved progress indicators** now appear in the chapter list so you can quickly spot partially read chapters
- **Resume from the saved page** when reopening a manga chapter instead of starting over

## 🔧 Improvements

#### ReadAllComics Compatibility Refresh
- Repaired the **ReadAllComics library feed** so content loads correctly again
- Fixed the **empty reader issue** affecting some ReadAllComics chapters
- Updated parsing to better handle the latest ReadAllComics page structure

## 🐛 Bug Fixes
- Sanitized image URLs and `Referer` headers to prevent loading issues caused by stray whitespace or newline characters

**Update now** to enjoy a smoother manga resume experience and more reliable ReadAllComics browsing! 📚✨

As always, if you encounter any issues, please report them in our issues channel. Happy reading! 🦋
