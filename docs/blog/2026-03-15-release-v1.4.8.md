# 📢 InkNest v1.4.8 Release Announcement

Hey everyone! 🎉 We're excited to announce **InkNest v1.4.8** is now available!

This release improves local comic file handling and fixes cache-related issues for a smoother reading experience.

<!-- truncate -->

## ✨ What's New

#### CBR/CBZ File Type Filters
- **File type filters in file picker** — Now you can easily filter and browse CBR and CBZ comic archive files when importing local comics
- Improved file discovery makes it easier to find and open your comic archives

## 🐛 Bug Fixes
- **Fixed stale cache issues** caused by parser updates — Content now refreshes properly when parsers are updated

**Update now** to enjoy better local comic file support and more reliable content loading! 📚✨

As always, if you encounter any issues, please report them in our issues channel. Happy reading! 🦋
